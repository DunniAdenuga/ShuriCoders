Green Shopping

Thanks for installing!

We built Green Shopping Chrome extension so that you can be environmentally conscious while enjoying shopping online. Below are the simple instructions for using the extension. Enjoy!

  • Help

Adding Items is really simple! First, shop on Amazon and place the items you would like to view into your cart. Don't forget to set the correct quantity as well. Once, ready, click on the extension icon and press Show CarbonPrint.
